The tracker module contains the Tracker class which can alternatively be imported directly from the gpu_tracker package.

class gpu_tracker.tracker.Tracker(sleep_time: float = 1.0, ram_unit: str = 'gigabytes', gpu_ram_unit: str = 'gigabytes', time_unit: str = 'hours', disable_logs: bool = False, n_join_attempts: int = 5, join_timeout: float = 10.0, kill_if_join_fails: bool = False, process_id: int | None = None)[source]ÔÉÅ

Runs a thread in the background that tracks the compute time, maximum RAM, and maximum GPU RAM usage within a context manager or explicit start() and stop() methods. Calculated quantities are scaled depending on the units chosen for them (e.g. megabytes vs. gigabytes, hours vs. days, etc.).

Variables:

  • max_ram (MaxRAM) ‚Äì Description of the maximum RAM usage of the process, any descendents it may have, and the operating system overall.

  • max_gpu_ram (MaxGPURAM) ‚Äì Description of the maximum GPU RAM usage of the process and any descendents it may have.

  • compute_time (ComputeTime) ‚Äì Description of the real compute time i.e. the duration of tracking.

Parameters:

  • sleep_time (float) ‚Äì The number of seconds to sleep in between usage-collection iterations.

  • ram_unit (str) ‚Äì One of ‚Äòbytes‚Äô, ‚Äòkilobytes‚Äô, ‚Äòmegabytes‚Äô, ‚Äògigabytes‚Äô, or ‚Äòterabytes‚Äô.

  • gpu_ram_unit (str) ‚Äì One of ‚Äòbytes‚Äô, ‚Äòkilobytes‚Äô, ‚Äòmegabytes‚Äô, ‚Äògigabytes‚Äô, or ‚Äòterabytes‚Äô.

  • time_unit (str) ‚Äì One of ‚Äòseconds‚Äô, ‚Äòminutes‚Äô, ‚Äòhours‚Äô, or ‚Äòdays‚Äô.

  • disable_logs (bool) ‚Äì If set, warnings are suppressed during tracking. Otherwise, the Tracker logs warnings as usual.

  • n_join_attempts (int) ‚Äì The number of times the tracker attempts to join its underlying thread.

  • join_timeout (float) ‚Äì The amount of time the tracker waits for its underlying thread to join.

  • kill_if_join_fails (bool) ‚Äì If true, kill the process if the underlying thread fails to join.

  • process_id (int | None) ‚Äì The ID of the process to track. Defaults to the current process.

Raises:

ValueError ‚Äì Raised if invalid units are provided.
